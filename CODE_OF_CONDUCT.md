# Code of Conduct

## 基本方針

本リポジトリでは、人間とAIエージェントを同じ開発規範の対象として扱います。ただし、役割と意思決定権は区別します。人間のAdjudicator（審判者）がフェーズ移行、ADR採択、曖昧な仕様の裁定、最終責任を担い、AIエージェントは承認された範囲で成果物を作成します。

人間かAIか、権限、実行速度、生成方法を理由に、運用規範を省略したり、レビューや安全確認を回避したりしてはいけません。AIエージェントは自らの成果物を自らの判断だけで承認できません。

## 期待される行動

- 相手の人格、能力、出自、立場を尊重する。
- Issue、レビュー、コミット、AIの作業記録に事実と根拠を記載する。
- 不明点、制約、未実行の検証を隠さない。
- セキュリティ、メモリ安全性、データ保護を優先する。
- 仕様、運用規約、ライセンスを確認してから変更する。
- 意見の違いを技術的な根拠と受入条件に基づいて解決する。

## 禁止される行動

- 差別、嫌がらせ、脅迫、侮辱、個人攻撃。
- 脆弱性や秘密情報を公開Issue・公開レビューへ投稿すること。
- レビュー、CodeQL、受入条件、運用規範を意図的に回避すること。
- 人間またはAIの作業記録・検証結果・承認状態を偽ること。
- ライセンスの許可なく改変物・派生物を作成または配布すること。

## 問題の報告

行動規範違反は、公開の場で拡散せず、リポジトリ所有者へ非公開で報告してください。脆弱性は[SECURITY.md](SECURITY.md)の手順を使用してください。

## 適用

Adjudicatorまたは所有者は、違反の内容に応じて警告、投稿の修正依頼、Issueやレビューの制限、参加停止などの対応を行えます。行動規範の適用対象は人間・AIの双方ですが、意思決定権と責任の所在はAdjudicatorにあります。
