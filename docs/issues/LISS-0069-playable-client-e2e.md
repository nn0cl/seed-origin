# LISS-0069: プレイアブルクライアントE2E受入

- Priority: critical

- Depends on: LISS-0059, LISS-0060, LISS-0061, LISS-0062, LISS-0063, LISS-0065, LISS-0066, LISS-0067, LISS-0068
